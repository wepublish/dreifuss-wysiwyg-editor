# dreifuss-wysiwyg-editor

## Getting started
### Installation 

First, clone the repo:

```
git clone git@github.com:wepublish/dreifuss-wysiwyg-editor.git
```

Then:
```
lerna bootstrap
yarn build
```

And start the dev environment:

```
yarn watch
```
