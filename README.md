# dreifuss-wysiwyg-editor

## Getting started
### Installation 

First, clone the repo:

```
git clone git@github.com:wepublish/dreifuss-wysiwyg-editor.git
```

Then:
```
lerna bootstrap
yarn build
```

And start the dev environment:

```
yarn watch
```

## Available Plugins

- link
- link-ui
- table
- table-ui
- image
- image-ui
- font
- font-ui
- media-embed
- emoji-picker
- characters-count
- find-replace
- find-replace-ui
- quotation-mark
- quotation-mark-ui

### Try it
You will find it working on `http://localhost:8080`
